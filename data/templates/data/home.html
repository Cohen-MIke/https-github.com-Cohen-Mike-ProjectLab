Abstract

Antibodies, expressed on the surface of B lymphocytes or secreted in serum, must offer enormous diversity in order to recognize and neutralize a large number of potential pathogens. Several factors contribute to antibody diversity, including the multiple germline gene segments that make up the primary repertoire. During an immune response, certain mechanisms make it possible to increase the diversity and the affinity to the antigen, namely somatic hypermutation and secondary rearrangements. The isotypic permutation process allows antibodies to change biological function while keeping their specificity for the antigen.
Our database is made up of a few datasets, including Celiac disease, tumor, and others ...
With the advances in high throughput sequencing technologies, we are able to sequence the antibody chains and form a repertoire. This repertoire is a representation of the antibodies within an individual. The antibodies play a key role in auto-immune disease, for example by using the sequenced repertoire one can differentiate between healthy and unhealthy individuals.

Goal of this interface

Today, for creating such a classifier you need to collect the data, engineer the features and then apply machine learning algorithm. Our goal is to give the possibility to a non-technological biologist to be able to access this system. To do this, this interface automate that system and is a tool that can pull data sets from existing servers, and that can build features from a known list the consumer request and create a machine learning algorithm that can be tweaked. The target clients for this tool is the b cell research community.
